# Supabase Quest

A simple 2D browser-based game (Flappy Bird style) featuring user authentication and a global leaderboard powered by **Supabase**.

## Features
- **User Authentication:** Sign up and log in using email and password via Supabase Auth.
- **2D Gameplay:** A classic canvas-based game where you dodge obstacles to score points.
- **Leaderboard:** Compete with others and see the top 10 high scores in real-time.
- **Persistent Progress:** Your personal high score is saved to the database.

## Tech Stack
- **Frontend:** Vanilla JavaScript, HTML5 Canvas, Tailwind CSS.
- **Backend:** Supabase (Auth & Database).

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. Configure your Supabase URL and Key in `app.js`.
